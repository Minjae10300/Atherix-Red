Atherix Red is an autonomous, multi-agent AI pentesting system built around a ReAct control loop. Specialized researcher, coder, and critic agents coordinate under an autonomous curiosity loop to explore, plan, and execute offensive security tasks in a self-learning lab environment. Under the hood, dual-backend LLM routing lets the system fall back between local Ollama models (Qwen) and cloud providers depending on the task, and a native tool-calling layer with a format-failure circuit breaker keeps the agents reliable when models drift off protocol. Agents have access to a tool registry that includes ARC-SIM, a multi-phase fusion reactor physics simulator (Lawson criterion, 0D ODE energy balance, 1D radial PDE, tokamak geometry), and SimForge, a Monte Carlo forecasting library. Everything in this repo is being built by one person from first principles as the core product of Atherix.
